Evaluation


After completing my project, I can proudly say that I am satisfied with my work. I’ve successfully met all of the Basic and Advanced requirements by:


(Basic Requirements)

  • choosing an appropriate dataset and cleaning the data using Pandas.
  • using Plotly to create labelled graphs, using lists to organise the countries into the correct continents under the variable name “continent_mapping” and storing statistical using a dictionary, “continent_stats”, to data for each continent for calculating the mean, mode, median and range for every continent.
  • creating a website, providing a navigation bar with appropriate titles that directs the user to the named pages and displaying the graphs on the “Statistics” page.

(Advanced Requirements)

  • creating the graphs using Plotly ensuring user can hover over the graph and filtering the continents by clicking one or more checkboxes.
  • creating a user poll using JavaScript, Python and HTML collecting string, boolean and integer values by asking the user to select their country, input whether they are happy or not and rate their happiness.
  • including suggestions, under the “recommendations” heading, of methods of how users can individually improve their wellbeing within their own countries.

An improvement I would make would be to focus on the design of the website. Unfortunately due to having no time, I was unable to design the website fully to my standard. This improvement would be beneficial as it draws the attention of the users. A second improvement would be to include my chosen dataset on the homepage of my website. I believe this would improve my project as it shows the user the original data displayed in the table.


Another application for my artefact would be to predict the possible future happiness scores using a regression model. This could allow for governments to view this website and consider how to implement change into society.
