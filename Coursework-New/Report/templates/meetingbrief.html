Meeting the Brief


How I met the Basic Requirements


Basic Requirement 1

  • I have chosen a suitable dataset with a sufficient amount of data to analyse
  • I used a Python library named “pandas” to successfully extract, clean and store the data, creating a new “cleaned” .csv file.

    • Basic Requirement 2

  • I used Plotly to display all of the results using bar, line, scatter and sunburst charts with an addition of a scatter plot.
  • I have labelled every graph with appropriate titles and axis labels. Legends are presented on the right of the bar charts, scatter plot and the pie chart.
  • I have used lists to organise the countries from my dataset into the correct continents under the variable name “continent_mapping”. I have used a dictionary with the variable name “continent_stats” to store statistical data for each continent that is useful for calculating the mean, mode, median and range for every continent.

    • Basic Requirement 3

  • I have created a website based on my chosen dataset. I have also provided a navigation bar with appropriate titles that directs the user to the named pages.
  • The data visualisations I have created are stored under “Statistics” that can be easily accessed by the user on the navigation bar mentioned.

    • How I met the Advanced Requirements


    Advanced Requirement 1

  • Plotly enables the user to hover over the graphs to view the details already. In addition, due to the continent_mapping variable I have created, it allows the user to filter the sunburst charts by clicking one or more checkbox buttons that is labelled after every continent.

    • Advanced Requirement 2

  • I have created a user poll under the “user poll” heading in the navigation bar. Here I collected string, boolean and integer values by asking the user to select their country, input whether they are happy or not and rate their happiness. After the user has submitted their results, they are then redirected to the “summary” page where they can see the results from all users displayed in a table. I have used JavaScript for the poll in the file titled “poll.js”.

    • Advanced Requirement 3

  • On the “summary” page, I have included suggestions, under the “recommendations” heading, of methods of how users can individually improve their wellbeing within their own countries.
