    Problem I Have Encountered


    One problem I have encountered with my code is the attempt of getting Flask to work properly. At first, when I would run the code in Thonny there would be no output displayed in the shell. I googled and was told I should be expecting a link to be shown that would redirect me to my website. I saw on an online forum on StackOverFlow that recommended installing waitress or gevent, which allowed me to believe that would be the solution, however this did not help. I then thought it was due to Flask being version 3.1.0 and Python was version 3.13.1. Therefore, I uninstalled the 3.13.1 version for Python and I installed 3.12.9 instead, making sure it is added to the system PATH. I then upgraded Flask to make sure it's installed in Python 3.12.9. I changed the Python executable in the tools, options and interpreter tab in Thonny to the correct directory. This allowed me to fix the problem regarding Flasks. Upon running the code again in Thonny, I then received the link. However, when I opened it, the website kept loading non-stop. This was the second problem I have encountered straight after. I found another forum on StackOverFlow that suggested killing all Python processes using “taskkill /F /IM python.exe”. Finally, I opened the website link again and it displayed the output HTML code I have typed in Visual Studio Code, demonstrating that it was working!
