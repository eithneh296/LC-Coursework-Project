Plan and Design


Description of Design


  1. For Basic Requirement 1, my chosen dataset, “Happiness Index Score” displays how happiness is measured across the world. The project is based on key factors such as GDP per capita, social support, healthy life expectancy, freedom to make life choices, generosity, and perceptions of corruption.
  2. The dataset will be stored on a CSV file (that can be accessed through Microsoft Excel) and will be extracted, cleaned and stored using pandas on Thonny, meeting Basic Requirement 1.
  3. A new Python program for Flasks will be created for developing my functioning website, meeting Basic Requirement 3.
  4. On the Flask program, I will use Plotly to create and display interactive graphs on my website. This will allow me to meet Basic Requirement 3 and Advanced Requirement 1.
  5. Upon opening the website, the user will be met with the “Home” page that explains the chosen dataset and the interactive information system.
  6. The website will contain an interactive navigation bar of every page that consists of:
    • “Statistics”
    • “User Poll”
    • “Summary”
    • “Home” (in most cases)
  7. On the “Statistics” page, all of the graphs will be displayed as well as tables of the calculated mean, mode, median and range of the data from every continent.
  8. On the “User Poll”, users will be met with 3 questions that they must answer before being redirected to the “Summary” page. The user poll will be created using Python, HTML and JavaScript. The 3 questions will collect data from string, boolean and integer values and stored in a new CSV file titled “user_poll”_data.csv”. These will allow me to meet Advanced Requirement 2.
  9. On the summary page, users will be met with a pie chart and recommendations to read from that will assist them in improving their happiness within their own countries, meeting Advanced Requirement 3.
